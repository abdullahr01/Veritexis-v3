Project Overview

Starkline is a powerful data dashboard designed to analyze uploaded CSV files and deliver in-depth statistical insights. It cross-compares performance across various departments, providing users with interactive tools to view different types of graphs and charts. In addition, the platform integrates an AI-powered chatbot that performs probabilistic analysis to offer tailored recommendations, empowering organizations to make informed, data-driven decisions—all built with React.js, Node.js, and Express.

Challenges

  • Handling potentially large CSV files without performance lags.
  • Ensuring smooth, real-time updates across various visualization tools.
  • Merging natural language processing with statistical recommendations.
  • Ensuring consistent, accurate comparisons across different datasets.

Solutions

  • Implemented stream-based parsing on the backend to process data in chunks, minimizing memory usage.
  • Utilized React’s state management along with optimized chart libraries to render graphs dynamically and responsively.
  • Integrated an AI-driven chatbot using custom Node.js middleware to interpret user queries and provide data-backed insights.
  • Built a modular analysis engine that normalizes data and performs comparative analytics across various departmental metrics.
